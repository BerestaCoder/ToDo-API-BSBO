# ToDo лист API

### Описание
Благодаря этому проекту можно управлять задачаси с использованием матрицы Эйхенхауэра
### Технологии
Pyrthon 3.11
fastapi 0.119
### Запуск проекта в dev-режиме
- Установите и активируйте виртуальное окружение 
- Установите зависимости из файла requirements.txt
```
pip install -r require,emts.txt
```
- В папке с файлом main.py выполните комманду:
```
uvicorn main:app --reload 
```
